# 🧠 Multi-Disease Prediction System

An AI-powered healthcare prediction system built using Machine Learning and Streamlit.

## Features

- Diabetes Prediction
- Heart Disease Prediction
- COVID-19 Prediction
- Risk Assessment
- Precaution Recommendations

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Random Forest
- Streamlit

## Project Structure

```text
app.py
models/
datasets/
notebooks/
screenshots/
```

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
streamlit run app.py
```

## Model Performance

| Disease | Accuracy |
|----------|----------|
| Diabetes | ~75% |
| Heart Disease | ~71% |
| COVID-19 | ~96% |
